How this NSE Stock Valuation Tool works — short explanation

This tool was built to give fast, simple valuations for Indian stocks using a conservative method. Here's what it does and where the numbers come from.

Data sources

  • Market prices: best-effort from NSE public endpoints. If blocked, fallback is used.
  • Fundamentals: primary from yFinance. If missing, the tool scrapes Screener.in for EPS and P/E.

Calculation in plain terms

The app uses:

  • Trailing EPS (or scraped EPS)
  • A conservative expected growth rate (default)
  • A fair P/E based on historical/industry ranges

Forward EPS × Fair P/E = Intrinsic Value. Then the tool applies a margin of safety (default 30%) to produce buy/sell ranges.

If you want quick checks, type a ticker like RELIANCE.NS into the tool. Always double-check before trading.
